CONTRE
======
**CONT** inumm **RE** weighting.

This ist the documentation for ``CONTRE``.
``CONTRE`` is a package for the  sofware framework ``basf2``.
It can be used to improve the Continuum MC samples by reweighting,
using off-resonace data.
